    1. Clone project from github.

                  
$ cd <your_directory>/

    2. Open project in your IDE (IntelliJ) from gradle.build or pom .xml

    3. Start test and register your name

    	    
$ cd <your_directory>/
Mac + Linux:
$ ./gradlew start
Win
$ gradlew.bat start

    • When test passes, move to next test.
